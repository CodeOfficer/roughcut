## Reveal.js Terminology (ALWAYS Use These)

**ALWAYS use reveal.js domain language:**
- **Slides** (not segments/sections/pages)
- **Horizontal slides** (main sequence, `---` delimiter)
- **Vertical slides** (nested optional content, `^^` delimiter)
- **Fragments** (step-by-step reveals within slide)
- **Transitions** (slide change animations)
- **Themes** (presentation styling)
- **Speaker notes** (presenter-only content)
- **Auto-animate** (smooth element transitions)

---

## Audio Sync Flow

```
1. Navigate to slide
2. Play audio for slide
3. Wait for audio to finish
4. Pause for @pause-after seconds (viewers can read content)
5. Advance to next slide
```

---

## Markdown Format (Option 3 - Minimalist)

```markdown
# Slide Title
@duration: 8s
@pause-after: 2s
@background: #1e1e1e
@transition: zoom

Content here...

- Point 1 @fragment
- Point 2 @fragment +2s

@audio: Narration here [2s] with pauses [3s] inline.

@playwright:
- Action: Click button
- Wait 2s
- Type: "text" + Enter
```

**See**: `/planning/revealjs-integration/format-option-3-minimalist.md` for full spec
